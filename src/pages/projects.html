Personal Projects

Applying my knowledge to the random interests of my life

Raspberry Pi Server

Spending Christmas money on single board computers is a privilege. I recently purchased a Raspberry Pi 5 to begin working on my own pi cluster. Currently, I have a working recursive DNS server for ad-blocking and more secure home internet. I am currently working to host my own VPN for future home projects.

My long-term vision for these servers is to create a robotics, AI, and ML development server that I can access remotely and use as a work horse to begin experimenting with my own ideas and innovations in the respective fields (granted, I'll need to buy much more hardware!). Particularly, I'm interested in soft robotics, and would like to be able to run FEA on autonomous, compliant robotic systems using non-metal effectors and manipulators to interact with the world. I believe that this is the ultimate solution to collaborative robotics. Big Hero 6 had a good point.
